## package install

"npm install" or "npm i"

## theme || styles

- Styled-componets
  npm i styled-components
  npm i @types/styled-componets

## conection

npm i axios

## routes

npm i react-router-dom

## structure folders

- Components
  [name-folder] - in pascalCase
  [name-file] - index.tsx
  [name-styles] - styles.ts

- Pages
  [name-folder] - in pascalCase
  [name-file] - index.tsx
  [name-styles] - styles.ts

## start application

npm start

## build application

npm run build

## test application

npm run test

## eject application

npm eject
